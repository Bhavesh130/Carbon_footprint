# Carbon Emission Prediction Website

A web application that uses machine learning to predict personal carbon emissions based on lifestyle factors.

## Features

- **Interactive Form**: User-friendly interface to input lifestyle data
- **Real-time Prediction**: Instant carbon emission calculation using ML model
- **Detailed Results**: Comprehensive breakdown of carbon footprint
- **Personalized Recommendations**: Actionable tips to reduce emissions
- **Responsive Design**: Works on desktop and mobile devices
